# 🤖 AI Interview Simulator

An interactive web application that simulates real interview scenarios and provides instant feedback with scoring.

---

## 🚀 Features

- 🎯 Dynamic interview questions
- 🧠 Smart answer evaluation (rule-based AI)
- 📊 Score generation (0–10)
- ⏱️ Real-time timer
- ✍️ Word count tracking
- ⏭️ Skip question functionality
- 📈 Final performance summary
- 💎 Modern dark UI with animations

---

## 🛠️ Tech Stack

- Backend: Python (Flask)
- Frontend: HTML, CSS, JavaScript
- Deployment: Render
- Version Control: Git & GitHub
